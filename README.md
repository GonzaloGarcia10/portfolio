# Portfolio — Gonzalo García

Pequeño portfolio estático generado a partir del CV proporcionado.

Archivos clave:
- `index.html` — página principal
- `styles.css` — estilos
- `script.js` — pequeñas mejoras de UX
- `assets/img/photo.jpg` — coloca aquí tu foto (sustituir imagen de ejemplo)

Cómo previsualizar:
- Opción 1: Abrir `index.html` directamente en el navegador.
- Opción 2 (recomendado si quieres evitar problemas CORS): usar un servidor local. Si tienes Python instalado, desde PowerShell en la carpeta del proyecto ejecuta:

```powershell
python -m http.server 8000
```

y abre `http://localhost:8000` en tu navegador.

Siguientes pasos recomendados:
- Reemplazar `assets/img/photo.jpg` por tu foto (tú me la puedes subir o arrastrar a la carpeta `assets/img`).
- Si quieres, puedo añadir:
  - Sección de proyectos con capturas y enlaces.
  - Botón para descargar el CV en PDF (sube el PDF y lo enlazo).
  - Versión en inglés u otras mejoras visuales.

Si quieres que haga algún ajuste de diseño (colores, tipografía, layout móvil), dime qué estilo prefieres y lo adapto.
